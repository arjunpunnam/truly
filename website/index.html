Up and Running in Minutes

Get started with Truly in three simple steps.

1

Define Your Schema

Import your data model from Swagger, JSON Schema, or define it manually. This becomes the foundation for your rules.

2

Build Your Rules

Use the visual builder to create IF-THEN rules. Combine conditions with AND/OR logic. Set actions like MODIFY, LOG, or WEBHOOK.

3

Test & Deploy

Test with sample data in the simulator. Once satisfied, your rules are ready. Execute via API or the built-in console.

# Quick start with Docker
docker run -p 8092:8092 -p 3000:3000 truly/rule-engine

# Or clone and run locally
git clone https://github.com/arjunpunnam/tries.git
cd tries/2025-12-23-truly/rule-engine
./start-dev.sh
